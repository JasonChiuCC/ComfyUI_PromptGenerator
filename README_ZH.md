# ComfyUI 提示詞生成器

一個全面的 ComfyUI 提示詞生成器，包含 17 個大類和 297 個主題。

## 安裝

1. 將此專案複製到 `ComfyUI/custom_nodes/` 資料夾
2. 重啟 ComfyUI

## 使用方式

1. 在 `JC Prompt Generator` 類別下找到節點
2. 每個類別都有 EN（英文）和 ZH（中文）版本
3. 選擇主題並設定批次數量
4. 將輸出連接到你的工作流程

## 功能

- ✅ 297 個獨特主題
- ✅ 雙語支援（英文/中文）
- ✅ 批次生成
- ✅ 隨機種子控制
- ✅ 全部類別合併節點
